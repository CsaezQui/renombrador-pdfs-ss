# Renombrador de PDFs – Seguridad Social

Aplicación de escritorio para renombrar automáticamente los PDFs de la Seguridad Social (RLC y RNT) extrayendo la Razón Social del documento.

**Formato del nuevo nombre:** `RazonSocial_RLC.pdf` / `RazonSocial_RNT.pdf`

## Descargar la aplicación

Ve a la pestaña **Actions** → selecciona la última ejecución → descarga el artefacto correspondiente:

- `RenombradorPDFs_SS_Mac` → contiene el `.dmg` para macOS
- `RenombradorPDFs_SS_Windows` → contiene el `.exe` para Windows

## Tipos de documento detectados

| Texto en el PDF | Sufijo |
|---|---|
| `RELACIÓN NOMINAL DE TRABAJADORES` | `_RNT` |
| `Recibo de Liquidación de Cotizaciones` | `_RLC` |
